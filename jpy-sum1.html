UNITS, DIMENSIONS AND MEASUREMENT

Complete Chapter 1 Summary | JEE Physics

1.1 Physical Quantity

A quantity which can be measured and by which various physical happenings can be explained and expressed in form of laws is called a physical quantity.

Examples: Length, mass, time, force, etc.

Non-examples: Happiness, sorrow (cannot be measured)

Physical quantity (Q) = Magnitude × Unit = n × u

Where:
n = numerical value
u = unit

n ∝ 1/u (inverse relationship)

Example

10 meter means a length which is ten times the unit of length 1 meter.

Here 10 is the numerical value and meter is the unit.

1.2 Types of Physical Quantities

Ratio (Numerical Value Only)

No unit required

  • Relative density
  • Refractive index
  • Strain

Exception: Angle (requires unit: degrees or radians)

Scalar (Magnitude Only)

No direction

  • Length
  • Time
  • Work
  • Energy

Follow ordinary laws of addition

Vector (Magnitude & Direction)

Has direction

  • Displacement
  • Velocity
  • Acceleration
  • Force

Follow vector laws of addition

JEE TIP: Some quantities like moment of inertia are neither scalar nor vector - they are tensors.

1.3 Fundamental and Derived Quantities

Type Definition Examples

NOTE: The choice of fundamental quantities is arbitrary. In mechanics, any three quantities can be chosen as fundamental.

1.8 Dimensions of Physical Quantities

Dimensions represent the powers to which fundamental quantities must be raised to express a given physical quantity.

Example: Force
Force = mass × acceleration
= mass × (length/time²)
= mass × length × time⁻²
[Force] = [MLT⁻²]

1.11 Applications of Dimensional Analysis

Finding Units

Replace M, L, T with fundamental units to find derived units

Example: Work = Force × Displacement

[W] = [ML²T⁻²]

CGS: g cm²/s² (erg)

SI: kg m²/s² (joule)

Finding Dimensions

Use formulas to find dimensions of constants

Example: Gravitational constant

F = Gm₁m₂/r²

[G] = [M⁻¹L³T⁻²]

Unit Conversion

n₂ = n₁ [M₁/M₂]ᵃ [L₁/L₂]ᵇ [T₁/T₂]ᶜ

Example: 1 N = 10⁵ dyne

Checking Equations

Principle of homogeneity: All terms must have same dimensions

Example: s = ut + ½at²

[L] = [LT⁻¹][T] + [LT⁻²][T²]

[L] = [L] + [L] ✓

Deriving Relations

Assume relation and find powers using dimensional analysis

Example: Time period of pendulum

T = k√(l/g)

1.12 Limitations of Dimensional Analysis

  • Non-uniqueness: Same dimensions for different quantities (work, energy, torque)
  • Dimensionless constants: Cannot determine values like 1/2, 2π, etc.
  • Limited to product functions: Cannot derive relations like s = ut + ½at² or y = a sin ωt
  • More than 3 variables: Cannot derive if more than 3 physical quantities involved
  • Same dimensions: Cannot derive if two quantities have same dimensions

1.13 Significant Figures

Rules for Counting Significant Figures

  1. All non-zero digits are significant: 42.3 (3 sig fig), 243.4 (4 sig fig)
  2. Zeros between non-zero digits are significant: 5.03 (3 sig fig), 5.604 (4 sig fig)
  3. Leading zeros are NOT significant: 0.543 (3 sig fig), 0.006 (1 sig fig)
  4. Trailing zeros ARE significant: 4.330 (4 sig fig), 343.000 (6 sig fig)
  5. Exponential notation: Numerical portion gives significant figures

1.14 Rounding Off

  • Digit < 5: Preceding digit unchanged (7.82 → 7.8)
  • Digit > 5: Preceding digit raised by one (6.87 → 6.9)
  • Digit = 5 followed by non-zero: Preceding digit raised by one (16.351 → 16.4)
  • Digit = 5 followed by zeros, preceding even: Unchanged (3.250 → 3.2)
  • Digit = 5 followed by zeros, preceding odd: Raised by one (3.750 → 3.8)

1.15 Calculations with Significant Figures

  • Addition/Subtraction: Report to least number of decimal places
  • Multiplication/Division: Report to least number of significant figures

1.16 Order of Magnitude

Power of 10 when number is expressed in scientific notation

  • Speed of light: 3 × 10⁸ m/s ≈ 10⁸ m/s (order = 8)
  • Electron mass: 9.1 × 10⁻³¹ kg ≈ 10⁻³⁰ kg (order = -30)

1.17 Errors of Measurement

Absolute Error

Δa = |aₘ - a|

Difference between true value and measured value

Mean Absolute Error

Δā = (|Δa₁| + |Δa₂| + ... + |Δaₙ|)/n

Arithmetic mean of absolute errors

Relative Error

Δā/aₘ

Ratio of mean absolute error to mean value

Percentage Error

(Δā/aₘ) × 100%

Relative error expressed as percentage

1.18 Propagation of Errors

Operation Formula Error Propagation
Addition x = a + b Δx = ±(Δa + Δb)
Subtraction x = a - b Δx = ±(Δa + Δb)
Multiplication x = a × b Δx/x = ±(Δa/a + Δb/b)
Division x = a/b Δx/x = ±(Δa/a + Δb/b)
Power x = aⁿ/bᵐ Δx/x = ±(nΔa/a + mΔb/b)

Quick Formula Sheet

Essential Formulas

  • Physical quantity: Q = n × u
  • Unit conversion: n₂ = n₁ [M₁/M₂]ᵃ [L₁/L₂]ᵇ [T₁/T₂]ᶜ
  • Dimensional formula: [MᵃLᵇTᶜ]
  • Principle of homogeneity: All terms must have same dimensions
  • Absolute error: Δa = |aₘ - a|
  • Mean absolute error: Δā = Σ|Δaᵢ|/n
  • Relative error: Δā/aₘ
  • Percentage error: (Δā/aₘ) × 100%
EXAM STRATEGY: Always check dimensional homogeneity in derivations. For error analysis, identify the quantity with maximum power as it contributes most to error. Practice unit conversions regularly.

Practice Problems

Problem 1: Dimensional Analysis

Find the dimensions of Y in the equation X = 3YZ², where X has dimensions of capacitance and Z has dimensions of magnetic field.

Solution:
[X] = [M⁻¹L⁻²T⁴A²] (capacitance)
[Z] = [MT⁻²A⁻¹] (magnetic field)
[Y] = [X]/[Z²] = [M⁻¹L⁻²T⁴A²]/[M²T⁻⁴A⁻²] = [M⁻³L⁻²T⁸A⁴]

Problem 2: Unit Conversion

Convert 1 MW power to a system having basic units: mass = 10 kg, length = 1 dm, time = 1 minute.

Solution:
[P] = [ML²T⁻³]
n₂ = n₁ [M₁/M₂]¹ [L₁/L₂]² [T₁/T₂]⁻³
= 10⁶ [1 kg/10 kg]¹ [1 m/0.1 m]² [1 s/60 s]⁻³
= 10⁶ × 0.1 × 100 × 216000 = 2.16 × 10¹² units

Problem 3: Error Analysis

The resistance R = V/I where V = 100 ± 5 volts and I = 10 ± 0.2 amperes. Find the percentage error in R.

Solution:
ΔR/R × 100 = (ΔV/V × 100) + (ΔI/I × 100)
= (5/100 × 100) + (0.2/10 × 100)
= 5% + 2% = 7%
