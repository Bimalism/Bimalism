Key concepts and formulas for exam preparation

Chapter 1

Sets, Relations and Functions

  • A set is a collection of distinct objects
  • Types of relations: reflexive, symmetric, transitive
  • Function composition: (f∘g)(x) = f(g(x))
  • One-to-one (injective) functions
  • Onto (surjective) functions
  • Binary operations and their properties
  • Cardinality of sets
  • Inverse functions and relations
  • Equivalence relations and classes
  • Cartesian product of sets
Chapter 2

Complex Numbers and Quadratic Equations

  • i² = -1 (definition of imaginary unit)
  • Complex number form: z = a + bi
  • Modulus: |z| = √(a² + b²)
  • Conjugate: if z = a+bi, then z̄ = a-bi
  • Quadratic formula: x = [-b ± √(b²-4ac)]/(2a)
  • Polar form: z = r(cosθ + isinθ)
  • Euler's formula: e^(iθ) = cosθ + isinθ
  • De Moivre's theorem: (cosθ + isinθ)^n = cos(nθ) + isin(nθ)
  • Cube roots of unity: 1, ω, ω² where ω = -1/2 + i√3/2
  • Argand plane representation
Chapter 3

Matrices and Determinants

  • Matrix multiplication is not commutative
  • Determinant of 2x2 matrix: ad - bc
  • Inverse of matrix: A⁻¹ = adj(A)/|A|
  • Elementary row operations
  • Types of matrices: symmetric, skew-symmetric, orthogonal
  • Rank of a matrix
  • System of linear equations: AX = B
  • Cramer's rule for solving systems
  • Eigenvalues and eigenvectors
  • Properties of determinants
Chapter 4

Permutations and Combinations

  • nPr = n!/(n-r)!
  • nCr = n!/[r!(n-r)!]
  • 0! = 1 (by definition)
  • Circular permutations: (n-1)!
  • Combinations with repetition: (n+r-1)Cr
  • Binomial coefficient: C(n, r) = C(n, n-r)
  • Pascal's identity: C(n, r) + C(n, r+1) = C(n+1, r+1)
  • Number of subsets of a set: 2^n
  • Division of identical items: (n+r-1)Cr
  • Derangements: !n = n!∑(-1)^k/k!
Chapter 5

Mathematical Induction

  • Base case: Verify for n = 1 (or smallest value)
  • Inductive hypothesis: Assume true for n = k
  • Inductive step: Prove for n = k+1
  • Strong induction: Assume true for all values ≤ k
  • Sum of first n natural numbers: n(n+1)/2
  • Sum of squares: n(n+1)(2n+1)/6
  • Sum of cubes: [n(n+1)/2]²
  • Divisibility proofs using induction
  • Inequality proofs using induction
  • Principle of mathematical induction
Chapter 6

Binomial Theorem

  • (a+b)^n = Σ C(n, k) a^(n-k) b^k
  • General term: T_{r+1} = C(n, r) a^(n-r) b^r
  • Middle term: when n is even: T_{(n/2)+1}
  • When n is odd: two middle terms
  • Greatest coefficient term
  • Binomial expansion for negative exponents
  • Binomial expansion for rational exponents
  • Approximations using binomial theorem
  • Sum of binomial coefficients: 2^n
  • Sum of alternating binomial coefficients: 0
Chapter 7

Sequences and Series

  • Arithmetic progression: a, a+d, a+2d,...
  • AP nth term: a_n = a + (n-1)d
  • AP sum: S_n = n/2[2a + (n-1)d]
  • Geometric progression: a, ar, ar²,...
  • GP nth term: a_n = ar^(n-1)
  • GP sum: S_n = a(1-r^n)/(1-r) for r≠1
  • Infinite GP sum: a/(1-r) for |r|<1
  • Harmonic progression: reciprocals of AP
  • Arithmetic mean: (a+b)/2
  • Geometric mean: √(ab)
Chapter 8

Limits, Continuity and Differentiability

  • Limit definition: lim_{x→a} f(x) = L
  • Standard limit: lim_{x→0} sin(x)/x = 1
  • L'Hôpital's rule for 0/0 or ∞/∞ forms
  • Continuity: lim_{x→a} f(x) = f(a)
  • Differentiability implies continuity
  • Derivative definition: f'(x) = lim_{h→0} [f(x+h)-f(x)]/h
  • Product rule: (uv)' = u'v + uv'
  • Quotient rule: (u/v)' = (u'v - uv')/v²
  • Chain rule: (f∘g)'(x) = f'(g(x))·g'(x)
  • Rolle's theorem and Mean Value Theorem
Chapter 9

Integral Calculus

  • Fundamental theorem of calculus
  • Integration by substitution
  • Integration by parts: ∫u dv = uv - ∫v du
  • Definite integral properties
  • Standard integrals: ∫x^n dx = x^(n+1)/(n+1) + C
  • ∫(1/x) dx = ln|x| + C
  • ∫e^x dx = e^x + C
  • ∫sin(x) dx = -cos(x) + C
  • ∫cos(x) dx = sin(x) + C
  • Area under curves using integration
Chapter 10

Differential Equations

  • Order: highest derivative in equation
  • Degree: power of highest order derivative
  • Variable separable method
  • Homogeneous differential equations
  • Linear differential equations
  • Integrating factor method
  • Exact differential equations
  • Second order linear differential equations
  • Complementary function and particular integral
  • Applications: growth/decay, Newton's law of cooling
Chapter 11

Coordinate Geometry

  • Distance formula: √[(x₂-x₁)² + (y₂-y₁)²]
  • Section formula: (mx₂+nx₁)/(m+n), (my₂+ny₁)/(m+n)
  • Midpoint formula: ((x₁+x₂)/2, (y₁+y₂)/2)
  • Slope of line: m = (y₂-y₁)/(x₂-x₁)
  • Equation of line: y - y₁ = m(x - x₁)
  • Distance from point to line: |ax₁+by₁+c|/√(a²+b²)
  • Conic sections: circle, ellipse, parabola, hyperbola
  • Circle equation: (x-h)² + (y-k)² = r²
  • Parabola equation: y² = 4ax
  • Ellipse equation: x²/a² + y²/b² = 1
