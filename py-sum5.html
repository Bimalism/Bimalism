1. Moment of Force & Angular Momentum

Torque (Moment of Force)

τ⃗ = r⃗ × F⃗
Magnitude: τ = r F sinθ
Unit: N·m | Direction: Right-hand rule

Angular Momentum (L)

For a particle: L⃗ = r⃗ × p⃗ = r⃗ × m v⃗
For rigid body (fixed axis): L = I ω

τ⃗ = dL⃗/dt
→ If net external torque = 0 → Angular momentum conserved

2. Conservation of Angular Momentum

If net external torque on a system is zero → Total angular momentum remains constant.
I₁ω₁ = I₂ω₂

Classic Examples

  • Ice skater spins faster by pulling arms (I ↓ → ω ↑)
  • Diver tucks body → rotates faster
  • Collapsing star → neutron star spins extremely fast
  • Person on rotating platform catches a ball

3. Centre of Mass (COM)

Position vector of COM:
R⃗_cm = (m₁r⃗₁ + m₂r⃗₂ + … + mₙr⃗ₙ)/(m₁ + m₂ + … + mₙ) = Σ(mᵢ rᵢ)/M
Properties
• Entire mass of system can be assumed at COM
• COM moves as if all external forces act on it
• v_cm = (Σ mᵢ vᵢ)/M | a_cm = F_ext / M

Two-Particle System

COM divides the line joining masses in inverse ratio:
m₁x₁ + m₂x₂ = (m₁+m₂)x_cm

5. Parallel & Perpendicular Axis Theorems

Parallel Axis Theorem

I = I_cm + M d²
(d = distance between parallel axes)

Perpendicular Axis Theorem (Planar bodies only)

I_z = I_x + I_y
(I_z ⊥ to plane of lamina)
